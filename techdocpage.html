HTML Overview

HTML (HyperText Markup Language) is the standard language used to create webpages. It defines the structure of web documents and consists of elements that the browser renders.

HTML elements are used to structure the page and define content like headings, paragraphs, links, images, and more.

HTML tags are enclosed in angle brackets, with an opening and closing tag.

Common HTML elements include:

  • <h1> for top-level headings
  • <p> for paragraphs
  • <a> for links
  • <img> for images
  • <div> for division blocks

Example of HTML syntax:

<h1>Welcome to HTML!</h1>

HTML5 is the latest version of HTML and comes with new features like the <video> and <audio> elements.

<!DOCTYPE html>

CSS Overview

CSS (Cascading Style Sheets) is used to control the layout and style of a webpage. It defines how HTML elements should be displayed on the screen.

CSS allows you to apply styles like colors, fonts, spacing, and positioning.

  • color: Specifies the text color
  • font-family: Sets the font for the page
  • background-color: Changes the background color
  • margin: Sets the margin around elements
  • padding: Adds space inside an element

Example of CSS syntax:

body { background-color: #f4f4f4; }

CSS can be applied inline, internally, or externally. External stylesheets are the most common practice.

link rel="stylesheet" href="styles.css"

JavaScript Overview

JavaScript is a programming language that enables interactive elements on a webpage. It is used to create dynamic content such as animations, form validation, and much more.

JavaScript is often used in combination with HTML and CSS to build interactive websites.

  • let: Used to declare variables
  • function: Defines reusable blocks of code
  • console.log(): Outputs data to the browser console
  • document.getElementById(): Accesses HTML elements
  • addEventListener(): Adds event listeners to elements

Example of JavaScript code:

console.log('Hello, World!');

JavaScript can be included in a webpage using <script> tags, or linked externally.

<script src="script.js"></script>

SQL Overview

SQL (Structured Query Language) is a programming language designed for managing and manipulating relational databases. It is used to interact with databases by querying, updating, inserting, and deleting data.

SQL allows you to manage databases efficiently, making it a critical skill for working with relational databases like MySQL, PostgreSQL, Oracle, and SQLite.

  • SELECT: Retrieves data from a database
  • INSERT INTO: Adds new data to a table
  • UPDATE: Modifies existing data
  • DELETE: Removes data from a table
  • CREATE: Defines new database structures such as tables and views

Example of a SQL query to retrieve data from a table:

SELECT * FROM users WHERE age > 30;

SQL queries can be executed directly through a database management system or via a programming language that supports database interaction.

SELECT column_name FROM table_name WHERE condition;

Python Overview

Python is a high-level, interpreted programming language known for its readability and simplicity. It's widely used in fields like web development, data analysis, machine learning, and automation.

Python emphasizes code readability and uses indentation to define code blocks, making it easy to write and maintain.

  • print(): Outputs text or data to the console
  • def: Defines a function
  • import: Imports external libraries or modules
  • if: Conditional statement
  • for: Loop through sequences

Example of a Python function:

def greet(name):
        print("Hello, " + name)

Python can be used for web development (with frameworks like Django or Flask), data analysis (using libraries like pandas and NumPy), and much more.

import numpy as np
